# 仿网易云移动端
## 技术栈 vue^3.2.13+betterscroll^2.4.2



## 后端接口地址

```shell
https://github.com/Binaryify/NeteaseCloudMusicApi
```



## 初始化项目
```
npm install
```

### 运行项目
```
npm run serve
```

### 打包项目
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).





## 功能

```
- 推荐歌单列表

- 歌手分类页面
  - 根据热门分类
  - 根据26个英文字母分类
  
- 歌手/歌单详情页

- 全局播放器
  - cd/歌词切换
  - 基本齐全的播放功能


- mini播放器

```
